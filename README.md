## Getting Started

```
- Create Postgresql database using provided schema inside database folder
- Update `.env` inside `backend` folder
- Run `npm install` inside backend folder
- Run `npm install` inside frontent > voicescript-dashboard folder
- Run `npm run dev` inside backend folder to start API Server
- Run `npm run dev` inside frontend > voicescript to start the Frontend
```
